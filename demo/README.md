## What's in the Demo

The demo repository contains a complete implementation of AI weather agents using the Aether SDK with x402 payment protocol on Solana:

- **Weather Provider Agent** - Sells weather data with x402 payment protection
- **Customer Agent** - Purchases weather data using USDC on Solana
- **Simple Agents** - Basic HTTP implementation for comparison
- **Complete Documentation** - Setup guides, payment flow explanations, and troubleshooting

## Quick Start

```bash
git clone https://github.com/AETHER-SDK/aether-sdk-demo.git
cd aether-demo
npm install
# Configure .env
npm run start:weather    # Terminal 1
npm run start:customer   # Terminal 2
```

See the [demo repository](https://github.com/AETHER-SDK/aether-sdk-demo) for full documentation.
